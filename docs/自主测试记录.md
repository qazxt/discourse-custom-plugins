## 追加：网站打不开（2026-03-22 排障）

**现象**：浏览器打不开站点；WSL 内 `curl http://127.0.0.1:4200/` → **`Connection reset by peer`**；`curl :3000` 仍 **200**。

**原因**：Docker 仍映射 `127.0.0.1:4200→容器:4200`，但容器内 **已无 Ember 监听 4200**（进程挂掉或只曾在 4202 等情况），端口能连上立刻被对端 RST。

**已执行修复**（root + `/root/discourse`）：

```bash
cd /root/discourse
nohup ./d/ember-cli > /tmp/ember-restart.log 2>&1 &
# 等到日志出现 Build successful 后再访问
tail -f /tmp/ember-restart.log
```

**复验**：`curl -sI http://127.0.0.1:4200/` → **HTTP 200**；容器 `/proc/net/tcp6` 可见端口 **1068**（即 4200）处于 LISTEN。

**日常**：若再出现 reset，优先 **只重启 Ember**（勿在容器内 `docker exec … ./d/ember-cli`）；仍异常可参考 `本地启动调试.txt` 里「网站打不开 / 4200 connection reset」整段（含 `pkill`、`docker restart discourse_dev`）。
